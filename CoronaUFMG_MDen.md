<p>b) a vari√¢ncia deveria aumentar com a magnitude dos dados.</p>
<h5><a id="user-content-caracter√≠sticas-de-interesse" class="anchor" aria-hidden="true" href="#caracter√≠sticas-de-interesse"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Caracter√≠sticas de interesse:</h5>
<p>As caracter√≠sticas mais importantes s√£o:</p>
<p><strong>1) Taxa de infec√ß√£o</strong></p>
<ul>
<li>$c$ mede a velocidade da acelera√ß√£o e reflete a taxa de infec√ß√£o da doen√ßa.</li>
</ul>
<p><strong>2) Ass√≠ntota</strong>
$$\lim_{t \to \infty}  \mu( t) = \lim_{t \to \infty} \frac{ a ., \exp{ { c t } } } {1 + b ., \exp { { ct } }} = \frac ab$$</p>
<ul>
<li>
<p>Reflete o total de casos acululados ao longo de toda a pandemia.</p>
</li>
<li>
<p>Crescimento exponencial ($b=0$): ass√≠ntota $= \infty$ !</p>
</li>
</ul>
<p><strong>3) Ponto de inflex√£o</strong></p>
<ul>
<li>
<p>√â o tempo $t*$ onde o n√∫mero de novos casos para de crescer e come√ßa a diminuir.</p>
</li>
<li>
<p>Crescimento exponencial ($b=0$): n√∫mero de novos casos nunca para de crescer!</p>
</li>
</ul>
<p><strong>4) Previs√£o:</strong></p>
<ul>
<li>O que podemos dizer sobre $Y (t+k ), \forall k$, para $t$ fixo (hoje)?
Depende da distribui√ß√£o de $Y( t)$ mas ser√° sempre dada pela distribui√ß√£o preditiva de $Y(t+k)$ dado $Y(1:t) = { Y( 1) , ... , Y( t ) }$, o que foi observado.</li>
</ul>
<p>Funciona como se fosse a distribui√ß√£o a posteriori de $Y(t + k )$.</p>
<p><strong>Resultado √∫til:</strong> Se $Z$ e $W$ s√£o 2 v. a.'s quaisquer ent√£o:</p>
<p>$E[Z] = E[ E(Z \mid W ) ]$</p>
<p>$Var[Z] = Var[ E(Z \mid W ) ] +  E[ Var( Z \mid W ) ]$</p>
<p>Em particular,
$E[Y ( t + k ) \mid Y( 1:t)] = E{ E[ Y ( t + k ) \mid \mu( 1:t)] \mid Y( 1:t) }$
$= E[ \mu( t+k )] \mid Y( 1:t) ]$, a m√©dia a posteriori de $\mu ( t + k )$.</p>
<p>A infer√™ncia para tudo que foi dito acima deve ser reportada atrav√©s de estimadores pontuais (ex: m√©dias a posteriori), junto com os respectivos intervalos de credibilidade.</p>
<p><strong>5) N√∫mero de reprodutibilidade $R_0$</strong></p>
<p>$R_0$ √© o n√∫mero esperado de casos secund√°rios de uma doen√ßa produzidos por um indiv√≠duo infectado.</p>
<p>No tempo $t$, ele √© definido como
$$R_0 = \frac {\mu ( t ) - \mu ( t-1)}{\mu ( t-1)}  = \frac {\mu ( t )}{\mu ( t-1)} - 1$$</p>
<p><strong>In√≠cio da epidemia:</strong>
$1 \gg b \exp { { ct } } \to \mu (t) \approx a \exp { { ct } } \to R_0 \approx e^c - 1$</p>
<p><strong>Final da epidemia:</strong>
$1 \ll b \exp { { ct } } \to \mu (t) \approx a / b \to R_0 \approx 0$</p>
<p><strong>Meio da epidemia:</strong>
$R_0$ √© fun√ß√£o dos par√¢metros $(a,b,c)$ e do tempo $t$, e √© dado por
$$R_0 (t) = e^c \ \frac{1 + b e^c e^{ct} } {1 + b e^{ct} } \ - \ 1$$</p>
<p>Para cada $t$ fixado, podemos obter sua distribui√ß√£o a posteriori (via amostra MCMC) e calcular m√©dia, quantis e intervalos de credibilidade.</p>
<p><strong>6) N√∫mero m√©dio de novos casos (NMNC)</strong></p>
<p>NMNC no tempo $t+k$:
$n_t ( k ) = E [Y ( t + k ) - Y ( t + k -1 ) ] = \mu ( t + k ) - \mu ( t + k -1 )$</p>
<p>Logo, NMNC tamb√©m √© fun√ß√£o dos par√¢metros $(a,b,c)$ e pode ser facilmente calculado.</p>
<p>Para cada $t$ e $k$ fixados, podemos obter sua distribui√ß√£o a posteriori (via amostra MCMC) e calcular m√©dia, quantis e intervalos de credibilidade.</p>

<h4><a id="user-content-alternativas" class="anchor" aria-hidden="true" href="#alternativas"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Alternativas</h4>

<h5><a id="user-content-poisson-com-sobredispers√£o" class="anchor" aria-hidden="true" href="#poisson-com-sobredispers√£o"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Poisson com sobredispers√£o</h5>

<p>1.4) $Y( t ) \mid \epsilon ( t ) \sim Poisson ( \mu ( t ) \times \epsilon ( t ) )$ com $E[  \epsilon (t)] = 1$ e $Var( \epsilon (t)) = \sigma^2$</p>
<p>Usando os resultados √∫teis anteriores:</p>
<p><strong>Mod(1.3):</strong>
$E [ Y ( t ) ] = E[ E( Y(t) \mid \epsilon (t ) ) ] = E[  \mu ( t ) + \epsilon ( t ) ] = \mu ( t ) + E[ \epsilon ( t ) ] = \mu ( t )$</p>

<p>$Var[ Y ( t ) ] =  Var[ E( Y(t) \mid \epsilon (t) ) ]  + E[ Var ( Y(t) \mid \epsilon (t) ) ] = Var[ \mu ( t ) \times \epsilon ( t )  ] + E [ \mu ( t ) \times \epsilon ( t ) ] =

<h5><a id="user-content-extens√µes-din√¢micas" class="anchor" aria-hidden="true" href="#extens√µes-din√¢micas"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Extens√µes din√¢micas</h5>
<p>Modelos anteriores assumem comportamento est√°tico $\Rightarrow$ a forma
da doen√ßa n√£o se modifica ao longo do tempo $\Rightarrow$ taxa de
infec√ß√£o ser√° sempre a mesma, assintota ser√° sempre a mesma, ...</p>
<p><strong>Modelos din√¢micos</strong> flexibilizam isso.</p>
<p>$$\mu ( t ) =  \frac{ a( {\color{red} t )} \ \exp{ { c( {\color{red}t) } \  t } } } {1 + b( {\color{red}t) } \ \exp { { c({\color{red}t) } \ t } }}$$</p>
<p>com:
$a ( t ) = a ( t-1) + w_a ( t )$, onde $w_a ( t ) \sim N ( 0 , W_a ), \forall t $.</p>
<p>$b ( t ) = b ( t-1) + w_b ( t )$, onde $w_b ( t ) \sim N ( 0 , W_b ), \forall t $.</p>
<p>$c ( t ) = c ( t-1) + w_c ( t )$, onde $w_c ( t ) \sim N ( 0 , W_c ), \forall t $.</p>
<p><strong>Vantagens:</strong></p>
<ol>
<li>
<p>$E[ a(t) \mid a(t-1 )]= a (t-1)$, e o mesmo vale para $b(t)$ e $c(t) \Rightarrow$ const√¢ncia local.</p>
</li>
<li>
<p>$Var[ a(t) \mid a(t-1 )]= W_a$, e o mesmo vale para $b(t)$ e $c(t) \Rightarrow$ aumento da incerteza.</p>
</li>
</ol>
<p><strong>Problemas:</strong></p>
<ol>
<li>
<p>vari√¢ncias $W_a, W_b, W_c$ conhecidas $\Rightarrow$ dif√≠ceis de especificar.</p>
</li>
<li>
<p>vari√¢ncias $W_a, W_b, W_c$ deconhecidas $\Rightarrow$ dif√≠ceis de estimar.</p>
</li>
<li>
<p>n√£o d√° para simplificar $W_a = W_b = W_c = W$ (magnitudes diferentes de $(a,b,c)$).</p>
</li>
</ol>
<ul>
<li>Outra forma de introduzir dinamismo, agora multiplicativo:</li>
</ul>
<p>$a ( t ) = a ( t-1) \times w_a ( t )$, onde $w_a ( t ) \sim Gamma ( d_a ,d_a ), \forall t $.</p>
<p>$b ( t ) = b ( t-1) \times w_b ( t )$, onde $w_b ( t ) \sim Gamma ( d_b ,d_b ), \forall t $.</p>
<p>$c ( t ) = c ( t-1) \times w_c ( t )$, onde $w_c ( t ) \sim Gamma ( d_c ,d_c ), \forall t $.</p>
<p><strong>Vantagens:</strong></p>
<ol>
<li>
<p>$E[ a(t) \mid a(t-1 )]= a (t-1)$ e o mesmo vale para $b(t)$ e $c(t) \Rightarrow$ const√¢ncia local.</p>
</li>
<li>
<p>$Var[ a(t) \mid a(t-1 )]= d_c^{-1}$ e o mesmo vale para $b(t)$ e $c(t) \Rightarrow$ aumento da incerteza.</p>
</li>
<li>
<p>Hiperpar√¢metros $d_a, d_b, d_c$ f√°ceis de especificar.</p>
</li>
</ol>
<p>Exemplos:
$$d=1000 \ \to \ 0,90= P ( 0,95 &lt; w(t) &lt; 1,05 ) = P \left( 0,95 &lt; \frac {a(t)}{a(t-1) } &lt; 1,05 \right)$$
$$d=1500 \ \to \ 0,95= P ( 0,95 &lt; w(t) &lt; 1,05 ) = P \left( 0,95 &lt; \frac {a(t)}{a(t-1) } &lt; 1,05 \right)$$</p>
<p><strong>Problemas:</strong></p>
<ol>
<li>
<p>As magnitures de $a, b, c$ ainda interferem no aumento da incerteza.</p>
</li>
<li>
<p>n√£o sei se software lida bem com Gammas tendo par√¢metros t√£o altos.</p>
</li>
</ol>
<p><strong>Evolu√ß√£o multiplicativa com erros normais</strong></p>
<p>Considere a evolu√ß√£o multiplicativa abaixo para o par√¢metro $a$:</p>
<p>$$a ( t ) = a ( t-1) \times \exp { w_a ( t ) }, \mbox{ onde } w_a ( t ) \sim N( 0 , W_a )$$
Tomando o logartimo nos dois lados, obtem-se:
$$\log \ a ( t ) = \log \ a ( t-1) + w_a ( t ), \mbox{ onde } w_a ( t ) \sim N( 0 , W_a )$$
Passando $\log \ a ( t-1)$ para a esquerda, vem que:
$$\log \ a ( t ) - \log \ a ( t-1) =  \log \left[ \frac{ a ( t )}{ a ( t-1)} \right] = w_a ( t ), \mbox{ onde } w_a ( t ) \sim N( 0 , W_a )$$</p>
<p>Especifica√ß√£o de $W_a$: pode-se pensar em percentual de incremento, como antes.</p>
<p>$$0,95 = P \left( 0,95 &lt; \frac {a(t)}{a(t-1) } &lt; 1,05 \right) = P ( - 0,05 &lt; w_a(t) &lt; 0,05 )$$
Isso implica $2 \sqrt{W_a} = 0,05$, que implica $\sqrt{W_a} = 0,025 \ \Rightarrow W_a = (0,025)^2$.</p>
<p>Mesma especifica√ß√£o vale para $W_b$ e $W_c$, pois dimens√µes de $b$ e $c$ n√£o importam.</p>
<p><strong>Caso particular</strong></p>
<p>Baseado em Gamerman, Santos e Franco (J. Time Series Analysis, 2013):</p>
<p>$$\mu ( t ) =  \frac{ a( {\color{red}t)} \ \exp{ { c \  t } } } {1 + b \ \exp { { c \ t } }}$$
$a ( t ) = a ( t-1) \times w_a ( t )$, onde $ w_a ( t ) \sim Beta, \forall t$.</p>
<p>Pode ser usado tamb√©m para crescimento exponencial ($b=0$).</p>
<p><strong>Vantagem:</strong></p>
<ul>
<li>Permite contas exatas, dispensando aproxima√ß√µes MCMC.</li>
</ul>
<p><strong>Desvantagem:</strong></p>
<ul>
<li>N√£o permite $b$ e $c$ din√¢micos.</li>
</ul>
<h4><a id="user-content-generaliza√ß√µes-da-log√≠stica" class="anchor" aria-hidden="true" href="#generaliza√ß√µes-da-log√≠stica"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Generaliza√ß√µes da log√≠stica</h4>
